简单 CRM 架构

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_crm'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rails_crm
```
